# Security

## Reporting a vulnerability

Please do not publish exploit details in a public issue.

Use GitHub's private vulnerability reporting feature for this repository when
available. If the report is not security-sensitive, open a normal bug report.

Nyxbyte 0.1 is an offline desktop application: it does not transmit telemetry,
connect to an account, or call a network service. Future connected features
will be opt-in and documented with their permissions.
